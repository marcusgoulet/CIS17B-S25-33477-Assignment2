# Library System

- This program is to simulate a library software that allows the user to modify a database full of books and users.
## Features
- Menu
    - Manage Books
      - Add New Books
      - View all Books
    - Manage Users
      - Add User
        - 1 for student 2 for faculty
      - View all Users
    - Manage Transactions
      - Borrow Book by ISBN
      - Return Book by ISBN
